# TTM Painting Website

Marketing website for TTM Painting, built with Vite + React + TypeScript.

## Getting Started

Requirements:

- Node.js 18+
- npm

Install dependencies and run locally:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
npm run preview
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
